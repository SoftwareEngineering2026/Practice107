Описание
Проект демонстрирует сбор логов и метрик микросервисного приложения с использованием стека:
Grafana Alloy – агент для сбора логов и метрик.
Loki – хранилище логов.
Mimir – хранилище метрик (совместим с Prometheus).
Grafana – визуализация.

Приложение – простой Flask-сервер, который считает количество запросов и выставляет эту метрику на эндпоинте `/metrics`.
